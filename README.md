# MediaPulse

MediaPulse is a modern analytics dashboard for YouTube channels, built with Next.js, React, and Material UI. It provides beautiful charts, statistics, and tables to help you visualize and understand your channel's performance.

## Features

- Channel analytics overview (subscribers, views, videos)
- Interactive bar charts with image labels
- Time range selection for analytics (3 days, week, month)
- Responsive layout for all devices
- Beautiful video table with thumbnails, views, comments, and published year
- Uses Material UI and Tailwind CSS for styling

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Run the development server**
   ```bash
   npm run dev
   ```
3. **Open your browser**
   Visit [http://localhost:3000](http://localhost:3000)

## Folder Structure

- `frontend/` — Next.js app (UI, charts, tables)
- `backend/` — Laravel API (if used)

## Tech Stack

- Next.js
- React
- Material UI
- Tailwind CSS
- Recharts
